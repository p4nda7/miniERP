✨ Überblick

Das Mini ERP System ist ein umfassendes Kassensystem, das speziell für Pizzabestellungen entwickelt wurde. Es bietet Funktionen zur Bestellverwaltung, Rechnungsstellung, Verkaufsanalyse und Lagerverwaltung.

⚙️ Installation

📥 Repository klonen

git clone https://github.com/dein-repo/mini-erp.git

🔍 Datei öffnen

Öffne index.html in einem Webbrowser.

💻 Verwendung

Nutzer auswählen: Kundenname eingeben und Bestellung starten.

Bestellung aufgeben: Pizzasorte, Menge und Preis auswählen.

Bestellung abschließen: Bestellung in der Historie speichern.

Rechnungen anzeigen: Rechnungen generieren und drucken.

Verkaufsanalyse nutzen: Verkaufsdaten und Lagerbestände im Admin-Bereich analysieren.

🛠 Funktionen

🔹 Nutzerverwaltung

Nutzer auswählen: Kundennamen eingeben und Bestellung starten.

Nutzer wechseln: Aktuelle Bestellung zurücksetzen und neuen Nutzer auswählen.

🔹 Bestellverwaltung

Bestellung hinzufügen: Neue Position zur Bestellung hinzufügen.

Bestellung abschließen: Bestellung speichern und Gesamtbetrag berechnen.

🔹 Rechnungsverwaltung

Rechnung generieren: Rechnung für Bestellung erstellen.

Rechnung drucken: Rechnung ausdrucken.

🔹 Verkaufsanalyse

Umsatzübersicht: Tägliche, wöchentliche, monatliche und gesamte Umsätze anzeigen.

Gewinnübersicht: Gewinnstatistiken anzeigen.

Top-Produkte: Meistverkaufte Produkte auflisten.

Durchschnittswerte: Durchschnittlicher Bestellwert und Gewinn berechnen.

🔹 Lagerverwaltung

Produktpflege: Produkte hinzufügen, bearbeiten und löschen.

Lagerbestandsanzeige: Aktuellen Bestand anzeigen und Warnungen bei niedrigen Beständen.

🔹 Zahlungsverwaltung

Zahlungsstatus aktualisieren: Zahlungen verwalten.

Offene Beträge: Summe offener Beträge anzeigen.

🤖 Technologien

HTML/CSS – Gestaltung der Benutzeroberfläche.

JavaScript – Logik und Interaktivität.

LocalStorage – Speicherung von Bestell- und Produktdaten im Browser.
